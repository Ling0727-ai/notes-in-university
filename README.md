## 📝 笔记格式规范

本仓库严格遵循统一的 Markdown 格式规范，确保内容的一致性和可读性：

### LaTeX 公式规范 ⚠️
- **行内公式**：使用单层 `$`，如 `$x^2$`
- **独立公式**：使用双层 `$$`，如 `$$\int_0^1 x dx$$`
- **分数表示**：统一使用 `\dfrac{分子}{分母}`
- **表格中绝对值**：使用 `\vert` 替代 `|`

### 标题层级规范 ⚠️
- **一级标题**：`#` 第X章/第X部分
- **二级标题**：`##` 一、二、三... 或 1. 2. 3. ...
- **三级标题**：`###` 1. 2. 3. ... 或 (1) (2) (3) ...
- **分隔线**：一级标题前添加双行分隔线，二级标题前添加单行分隔线

### 特殊内容块规范 ⚠️
- **定义块**：`> [!definition]` - 所有概念定义必须使用此格式
- **定理块**：`> [!theorem]` - 重要定理
- **例题块**：`> [!example]` - 典型例题
- **证明块**：`> [!proof]` - 定理证明
- **注意块**：`> [!note]` - 注意事项

### 表格与图片规范
- **表格对齐**：默认使用居中对齐 `:---:`
- **图片引用**：使用 `![[路径|参数]]` 格式
- **资源存储**：图片统一存放在 `assets/学科名称/` 目录下
